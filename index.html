lrn

A community driven, command-line tutorial platform that gives a more personalized learning experience. Benefit from one of the interactive lessons or share your mastery of a subject with someone else.





Installation


LRN is easy to install via pip. Run the following command in your terminal and it will set up itself. 

$ sudo pip install lrn


Find a tutorial

Getting started is really simple. just input the following command and take a look at the different features lrn offers. 

$ lrn


Contributing

As an open source tool, lrn relies on the support of other developers. If you have any suggestions, contributions or ug fixes let us know! Fork our repo and make some changes.



Create a lesson

Unfortunately we don't know everything, we need your help to make this as comprehensive of a tool as possible. If you have mastered a skill/language/program, take a look at the documentation on our repo page and set up a tutorial for those less knowledgable! A command line tutorial can really benefit a self-taught developer.
