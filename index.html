LRN

A community driven, command-line tutorial platform that gives a more personalized learning experience. Benefit from one of the interactive lessons or share your mastery of a subject with someone else.





Installation


LRN is easy to install via pip. Run the following command in your terminal and it will set up itself. 

$ sudo pip install lrn
